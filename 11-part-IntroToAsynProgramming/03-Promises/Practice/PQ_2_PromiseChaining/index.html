Write two functions, fetchUserData and fetchUserPosts, that simulate fetching data from a server using Promises and demonstrate Promise chaining.

  1. fetchUserData(userId):
    • Takes a userId as an argument.
    • Returns a Promise that resolves after 2 seconds with an object containing userId and name.
    • If userId is missing, it should reject with an error message "User ID is required.".
  2. fetchUserPosts(userId):
    • Takes the same userId as an argument.
    • Returns a Promise that resolves after 1 second with an array of posts for that user.
    • If userId is missing, it should reject with an error message "User ID is required for fetching posts.".
